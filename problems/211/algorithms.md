## Trie

Store words in a Trie, use one node for each letter.

**Solutions:**

* [Javascript](javascript/211.js)

**Complexity:**

N = word length

* Time:
  * add: O(N)
  * search: O(N)

* Space:
  * add: O(N)
  * search: O(1)
